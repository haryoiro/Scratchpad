## 必要構成

* Node.js 10+
* yarn

## 始め方

### インストール

```
$ yarn install
```

### アプリケーションを開始

```
$ node index.js

or

$ yarn run start
```

### ポート番号の指定

```
$ export PORT=80
```

## 開発

```
$ yarn run build
```
